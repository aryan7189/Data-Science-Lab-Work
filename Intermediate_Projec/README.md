# Heart Disease Analysis using Exploratory Data Analysis (EDA)

## Description

This project focuses on analyzing a heart disease dataset to identify patterns, relationships, and key factors that contribute to heart disease. The analysis is performed using exploratory data analysis (EDA) techniques.

## Dataset

* Source: UCI Machine Learning Repository
* The dataset contains medical attributes such as age, cholesterol, blood pressure, heart rate, and other health indicators.
* It is used to study the presence and severity of heart disease.

## Steps Performed

1. Data Cleaning

   * Handled missing values
   * Checked data types and structure

2. Exploratory Data Analysis (EDA)

   * Analyzed distributions of numerical features
   * Studied relationships between variables

3. Data Visualization

   * Histograms for distribution analysis
   * Box plots for detecting outliers
   * Correlation heatmap for feature relationships
   * Bar chart for target variable distribution

## Results

* Age, cholesterol, and maximum heart rate show noticeable relationships with heart disease.
* Some features exhibit moderate correlation with the target variable.
* Outliers were identified in features like cholesterol and oldpeak.
* The dataset shows variation in heart disease severity levels.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Conclusion

EDA helps in understanding the structure of the dataset and identifying important features that influence heart disease. This analysis can be used as a foundation for building predictive machine learning models.
